Сервис для мониторинга вилочных погрузчиков «Силант»:

- гостевой поиск машины по заводскому номеру;
- личный кабинет с доступом к технике, ТО и рекламациям;
- разграничение прав по ролям (клиент, сервисная организация, менеджер);
- управление справочниками и API-документацией для менеджера.

## Технологии

- **Backend:** Django, Django REST Framework, SimpleJWT, django-allauth, drf-spectacular  
- **Frontend:** React, Vite, TypeScript  
- **База данных:** PostgreSQL  
- **Продакшен-статика:** WhiteNoise  
- **Контейнеризация:** Docker + docker compose
